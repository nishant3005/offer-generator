# offer-generator
A full stack web-app where users can generate different kinds of letter like full-time offer, resume, etc.

### To visit offer-generator click [here](https://offer-generator.netlify.app/)
## Features:

* User authentication(using Google OAuth)
* Can generate full-time offer letter, resume, etc.
* Resumes are ATS(Applicant Tracking System) friendly.
* History of every piece of letter you've generated.
* Users can edit already created offer letter and resumes.
* Letters are downloadable.

## How to run on your system?

* Clone the repo.
* Inside Frontend->src->index.js line no. 9 just change the clientId and put     
  yours(taken from Google Developer console).
* Change the hosted backend URL from everywhere in the Frontend and replace it with   yours.
* In Backend, add .env file and put your mongoDB cloud url.
* Install the node modules by running $npm install on the terminal
* Run npm start and you are good to go!!

## Tech-stack used:
 * React
 * Node.js
 * Express
 * MongoDB
 * React OAuth2|Google
 * JWT-Decode
